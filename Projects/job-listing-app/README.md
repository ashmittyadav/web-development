# 💼 Job Listing App

A modern and responsive Job Listing UI built with **React**.

---

## 📸 Preview

![Preview](preview.png)

---

## ✨ Features

- Reusable React Components
- Dynamic Rendering using `.map()`
- Responsive Card Layout
- Clean and Modern UI
- Company Logos
- Job Details
- Save Button UI
- Apply Now Button

---

## 🛠️ Tech Stack

- React
- JavaScript
- CSS
- Vite

---

## 📁 Folder Structure

```
src
│
├── components
│   └── Card.jsx
│
├── data
│   └── jobs.js
│
├── App.jsx
└── main.jsx
```

---

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/yourusername/job-listing-app.git
```

Go inside the folder

```bash
cd job-listing-app
```

Install dependencies

```bash
npm install
```

Run

```bash
npm run dev
```
